# CS7641 Machine Learning — Semester Workspace

This repository contains all materials, experiments, reports, datasets,
notes, and reusable infrastructure for Georgia Tech OMSCS CS7641.

## Structure

- 00_admin → Course logistics and policies
- 01_notes → Lecture and concept notes
- 02_readings → Textbooks and supplemental readings
- 03_assignments → Assignment tracking and deliverables
- 04_projects → ML experimentation and code
- 05_reports → LaTeX reports and final writeups
- 06_datasets → Raw and processed datasets
- 07_experiments → Experiment tracking and failed runs
- 08_reusable_code → Shared utilities and ML helpers
- 09_exam_prep → Study guides and formula sheets
- 10_templates → Reusable notebook/report templates
- 11_research_papers → Important ML papers
- 12_meeting_notes → Office hours and discussion notes
- 13_archive → Old versions and archived work

## Recommended Stack

- Python 3.11+
- scikit-learn
- PyTorch
- Optuna
- MLflow
- Polars + Pandas
- Plotly / Matplotlib
- JupyterLab
- Overleaf + LaTeX

## Workflow Philosophy

This course should be treated like:
- a research lab
- an ML engineering practicum
- a technical writing course

Focus on:
- reproducibility
- experiment tracking
- strong analysis
- clear communication
